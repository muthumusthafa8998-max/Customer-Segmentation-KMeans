# Customer Segmentation using K-Means

This project applies K-Means clustering to segment customers based on their purchasing behavior.

## Dataset Features
- Age
- Annual Income (k$)
- Spending Score (1-100)

## Steps Performed
1. Data cleaning
2. Outlier detection
3. Feature scaling using StandardScaler
4. Elbow method to determine optimal clusters
5. K-Means clustering
6. Visualization of customer segments

## Tools Used
Python  
Pandas  
Scikit-learn  
Matplotlib  

## Result
Customers were segmented into 4 groups based on income and spending behavior.
